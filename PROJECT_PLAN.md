# PROJECT_PLAN.md

1. 프로젝트: 심평원 의료 통계 기반 수익형 건강 정보 웹사이트
2. 목표: 공공데이터포털(심평원) API에서 질병 통계 및 진료비 정보를 수집하고, Gemini AI가 E-E-A-T 기준에 맞춰 매일 초고품질 건강 블로그 글을 자동 작성하며, Google AdSense로 수익화한다.
3. 기술 스택: Next.js (App Router), TypeScript, Tailwind CSS
4. 자동화: GitHub Actions (매일 자동 수집/작성), Cloudflare Pages (자동 배포)
5. 필수 룰: 모든 코드는 구글 검색엔진 최적화(SEO) 및 코어 웹 바이탈을 최우선으로 고려하여 작성한다.
