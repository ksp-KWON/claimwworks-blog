# Claimworks Blog - Agent Guideline

## 💡 개발 명령 가이드
- **빌드 테스트:** `cmd /c npm run build` (PowerShell 권한 에러 회피용 cmd 실행)
- **로컬 서버:** `cmd /c npm run dev`

## ✍️ 블로그 글쓰기 규칙 헌법 (Strict Rules)
1. **H1 사용 금지:** `# 제목` 대신 Frontmatter에만 제목 기입.
2. **소제목 박스 스타일 (H2 사용):** 주요 장의 큰 소제목은 반드시 `##`로 작성 (둥근 가로 꽉 찬 상자로 자동 렌더링).
3. **단어 간격 콜론 규칙:** 본문 내 모든 콜론(` : `)은 앞뒤로 한 칸씩 띄어 쓰기 예외 없음 (예: `주제 : 상세내용`).
4. **보상금 계산기 위젯 필수 삽입:**
   - 교통사고/맥브라이드/배상책임 주제: `<calculator type="auto" />` 단 한 번 삽입.
   - 실비보험/병원비 주제: `<calculator type="medical" />` 단 한 번 삽입.
5. **강조 색상 다변화:** `<red>`, `<orange>`, `<green>`, `<blue>`, `<purple>` 골고루 사용하여 강조.
6. **자동 내부 링크:** 기존 포스트 데이터베이스를 참고하여 본문 중 자연스럽게 1~2개 링크 연결.
7. **자가진단 리스트:** `## [🛡️ 내 보험금/보상금 1분 자가진단 체크리스트]` 아래에 `- [ ] ☑️ ...` 형태로 3~5개 체크박스 작성.
8. **자주 묻는 질문 FAQ:** `## [💡 자주 묻는 질문 (FAQ) TOP 3]` 작성 후 아래에 `### Q1. 질문` 형식으로 3가지 질문 및 답변 구성.
9. **상담 유도 CTA 배너:** `## [📞 교통사고 보상, 전문가의 도움으로 권리를 찾으세요]` 아래 카카오톡 무료 상담 링크(`https://open.kakao.com/o/sWeszp7`) 삽입.
10. **SEO 요약문:** 글의 맨 마지막 줄에 `[SEO_SUMMARY]: 요약글`로 끝맺기.
